注册Netlify

第一个选项就是 Github 登录, 一路确认就好了

绑定仓库

Github 认证登录过后，右边应该有个 Site from git. 点击进去一路确认，选择你的 blog 源代码仓库. 没记错之后会自动跳到部署页面
