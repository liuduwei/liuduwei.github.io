grub rescue 修复引导

1.进入 normal 模式:

  • set

  • ls (hdx,y)/boot/grub/

  • set root=(hdx,y)

  • set prefix=(hdx,y)/boot/grub/

  • insmod normal

  • normal
    2.启动 arch 重装 grub:

  • grub-install –target=x86_64-efi –efi-directory=/boot/EFI –bootloader-id=GRUB

  • grub-mkconfig -o /boot/grub/grub.cfg

3.重启结束

更换主题

直接下载主题，解压进入目录执行:

sudo ./install.sh
